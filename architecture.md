# System Architecture

The system follows a modular architecture that includes Business, Information System (IS), and Technology layers. :contentReference[oaicite:7]{index=7}

## Business Architecture
- Defines user roles and workflows  
- Supports interactions among faculty, students, and administrators  

## Information System Architecture
- Manages academic data flow  
- Connects syllabus, assignments, chatbot, and feedback modules  

## Technology Architecture
- Built with Next.js, FastAPI/Node APIs  
- Integrates Gemini or LM Studio AI models  
- Ensures security, encryption, and scalable performance
