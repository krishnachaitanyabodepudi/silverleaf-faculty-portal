<div class="content-section">
  <h1 class="section-title">Proposed Solution</h1>
  
  <p>The <strong>AI Faculty Support Portal</strong> is a centralized solution that assists faculty, students, and administrators with teaching workflows, academic communication, AI-based learning support, and grading assistance.</p>
  
  <h2 class="subsection-title">Core Components</h2>
  
  <div class="info-box">
    <h3 class="subsubsection-title">1. Course-Aligned AI Assistant</h3>
    <p>AI chatbot trained on:</p>
    <ul>
      <li>Syllabi</li>
      <li>Rubrics</li>
      <li>Reading material</li>
      <li>Course outcomes</li>
    </ul>
    <p>Supports:</p>
    <ul>
      <li>Class preparation</li>
      <li>Example generation</li>
      <li>Assignment guidance</li>
      <li>Clarification of concepts</li>
    </ul>
  </div>
  
  <div class="info-box">
    <h3 class="subsubsection-title">2. AI Feedback Analyzer</h3>
    <ul>
      <li>Uses Gemini API</li>
      <li>Analyzes PDF submissions</li>
      <li>Extracts context</li>
      <li>Generates preliminary feedback</li>
      <li>Helps faculty save grading time</li>
    </ul>
  </div>
  
  <div class="info-box">
    <h3 class="subsubsection-title">3. Assignment Workflow System</h3>
    <ul>
      <li>Admin assigns courses to faculty</li>
      <li>Faculty create assignments</li>
      <li>Students upload PDF submissions</li>
      <li>AI & faculty feedback stored in the portal</li>
    </ul>
  </div>
  
  <div class="info-box">
    <h3 class="subsubsection-title">4. Secure, Compliant AI Infrastructure</h3>
    <p>Supports:</p>
    <ul>
      <li>Local LM Studio (Llama models / Private inference)</li>
      <li>Cloud Gemini API (High-power inference)</li>
    </ul>
  </div>
</div>
