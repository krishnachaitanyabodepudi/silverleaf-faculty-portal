# Data Structure

Key entities include:  
- **Admin Role:** Course assignment and role creation  
- **Faculty Data:** Profiles, course info, assignment creation, AI access  
- **Assignments & Submissions:** Collected PDFs with secure text extraction  
- **AI Interaction:** Chatbot, feedback analyzer, logs  
- **Generated Outputs:** Draft feedback, annotated files  
:contentReference[oaicite:8]{index=8}

Data relationships follow a structured schema enabling accurate querying, compliance, and traceability. The workflow cycle from Admin → Faculty → AI explains how assignments move through preparation, submission, and feedback stages.
