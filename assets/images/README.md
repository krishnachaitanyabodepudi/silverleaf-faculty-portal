# Images Directory

Place your project images here:

- `logo.png` or `logo.svg` - Project logo
- `enterprise-architecture.png` - Enterprise architecture diagram
- `er-diagram.png` - Entity Relationship diagram
- `data-flow.png` - Data flow diagram
- `dashboard.png` - Dashboard screenshot
- `ai-chat.png` - AI assistant interface screenshot
- `syllabus.png` - Syllabus management screenshot
- `timetable.png` - Class timetable screenshot

## Image Requirements

- **Format**: PNG, JPG, or SVG
- **Recommended Size**: 
  - Logos: 200x200px to 400x400px
  - Diagrams: 1200x800px or larger
  - Screenshots: 1920x1080px or larger
- **Optimization**: Compress images for web use to improve page load times
